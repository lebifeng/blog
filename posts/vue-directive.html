# VUE 自定义指令

# 1.什么是指令? 什么是自定义指令?

v-if v-for v-model等提供某些特殊功能的属性就是指令, 我们可以使用组件中的directives:{}定义局部自定义指令, 或是Vue .directive ()来定义全局自定义指令

# 2.自定义指令有什么用?

用处多了, 自定义指令提供了一系列钩子函数(bind, inserted, update, componentUpdated, unbind ), 在这些钩子函数中, 可以实现对元素的任何操作

# 用法

下面演示如何使用自定义指令来做权限控制:

需求: 根据当前用户角色信息, 分别控制不同权限元素的展示与隐藏

<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <title>Title</title>
 <script type="text/javascript" src="../node_modules/vue/dist/vue.js"></script>
</head>
<body>
<div id="root">
 <index v-has:roles.any="['roleOne', 'roleTwo']"></index>
</div>
</body>
<script type="text/javascript">

 function removeChild(el) {
   el.parentNode.removeChild(el)
 }

 // 定义一个v-has指令
 Vue.directive('has', {
   inserted(el, entities, vNode) {
     try {
       let vm = vNode.componentInstance

       // v-has:arg.modifier = value
       let arg = entities.arg
       let value = entities.value
       let modifier = entities.modifiers
       
       if (!arg || (value && value.length === 0) || JSON.stringify(
           modifier) === '{}') {
         removeChild(el)
       }

       if (arg === 'roles') {

         // 如果是both, 则判断value内元素是否都在roles中
         if (modifier['both'] === true && value.every(e => {
           return vm[arg].includes(e)
         })) {
           console.log('i have both permissions')

           // 否则只要有一条符合就通过
         } else if (modifier['any'] === true && value.some(e => {
           return vm[arg].includes(e)
         })) {
           console.log('i have at least one of the permissions')
         } else {
           removeChild(el)
         }
       } else {

         // 不仅仅是role, 可以增加一些其他的arg, 这里不再展示
         removeChild(el)
       }
     } catch (e) {
       console.error(e)
     }
   }
 })

 Vue.component('index', {
   inheritAttrs: false,
   data() {
     return {
       roles: ['roleOne']
     }
   },
   template:
         `
     <p>我显示了! 我有相应权限!</p>
   `
 })
 let vm = new Vue(
     {
       el: '#root'
     }
 )
</script>
</html>

执行

当使用v-has:roles.any="['roleOne', 'roleTwo']", data的roles数组中有'roleOne', 'roleTwo'任意
一条,p标签都可以显示
当使用v-has:roles.both="['roleOne', 'roleTwo']", data的roles数组必须同时有'roleOne', 'roleTwo', 
p标签才可以显示

相关方法参阅VUE API 自定义指令
