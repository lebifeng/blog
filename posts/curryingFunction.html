# 函数柯里化

# 什么是柯里化?

柯里化是个动词, 在计算机科学中, 表示将一个接收多个参数的函数转化成只接收一个参数并且 能够处理剩余参数的函数, 简单理解就是柯里化后的函数, 每次只接受一个参数, 每次调用需要将剩余参数依次 传递, 直到所有参数收集完毕, 函数才执行, 并且返回结果

eg:

function add(a, b, c) {
  return a + b + c
}

// add函数 柯里化后的函数
function curryingAdd(a) {
  return function (b) {
    return function (c) {
      return a + b + c
    }
  }
}

// 执行

add(1, 2, 3) // 6
curryingAdd(1) // [Function]
curryingAdd(1)(2) // [Function]
curryingAdd(1)(2)(3) // 6

是不是理解了? add函数接收3个参数, 我们通过特殊手段获得add函数柯里化后的函数curryingAdd, curryingAdd每次只接收一个参数, 并且执行3次才最终返回结果, 前面两次执行是一个参数收集过程. 那么这个特殊手段是什么呢,我们怎么才能将一个函数柯里化呢?

# 通用柯里化函数

/**
 * 柯里化函数
 */
function curryfy(fn, args) {
  let len = fn.length
  args = args || []
  // 收集参数的过程
  if (args.length < len) {
    return function () {
      return curryfy(fn, args.concat(...arguments))
    }
  }
  // 参数收集完成直接执行
  return fn.apply(this, args)
}

let curryingAdd = curryfy(add)
curryingAdd(1)(2)(3) // 6
curryingAdd(1)(2, 3) // 6
