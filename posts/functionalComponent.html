# VUE函数式组件

# 1.VUE中， 函数式组件指什么？

引用vue官方的话

没有管理任何状态，也没有监听任何传递给它的状态，也没有生命周期方法，实际上，它只是一个接受一些 prop 的函数。在这样的场景下，我们可以将组件标记为 functional，这意味它无状态 (没有响应式数据)，也没有实例 (没有 this 上下文)

我的理解 参考React.js无状态组件：

let FunctionalComponent = (props) => (
    <div>{props.xxx}</div>
)

这样好理解了吧，就是像函数一样接收一些props，返回简单模板，不涉及状态管理等，合理使用函数式组件能提高页面性能

# 2.VUE中函数式组件用法

1.Vue.component

Vue.component('anchored-heading', {
    // 加上这个属性，表示该组件为functionalComponent
    functional: true,
    // functionalComponent因为没有实例， 所以这里面千万别像之前一样用this，
    // render函数提供了第二个参数 context,里面能拿到一些相关信息，具体参考VUE API
    render: function (createElement, context) {
      // 创建 kebab-case 风格的 ID
      let headingId = getChildrenTextContent(context.slots().default)
      .toLowerCase()
      .replace(/\W+/g, '-')
      .replace(/(^-|-$)/g, '')
      return createElement(
          'h' + context.props.level,
          [
            createElement('a', {
              attrs: {
                name: headingId,
                href: '#' + headingId
              }
            }, context.slots().default)
          ]
      )
    },
    props: {
      level: {
        type: Number,
        required: true
      }
    }
  })

# 注意： Vue.component定义函数式组件时，必须提供render函数，否则会报错

2.Vue单文件组件形式, 要求2.5.0版本以上

<template functional>
  <div class="cell">
    <div v-if="props.value" class="on"></div>
    <section v-else class="off"></section>
  </div>
</template>

<script>
export default {
  props: ['value']
}
</script>

注意事项:

  1. template中获取props的写法
  2. 函数式组件被ref引用, 得到的结果是HTMLElement
