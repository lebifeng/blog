# 常见前端页面优化手段

前端页面性能优化总体思路是 优化http请求 和 异步加载

# 优化http请求

  1. 使用CDN缓存服务器, 加速资源下载
  2. 合理使用http缓存(强缓存,协商缓存)
  3. 使用压缩后的资源(*.min, css sprites, css nano)
  4. 合理利用浏览器并发下载机制
  5. 优化js和css文件摆放位置(css样式放置在head中, js脚本放置在body后)
  6. 按需引入(例如使用ajax不要将jquery.js全部引入)

# 异步加载

  1. 合理使用async和defer属性(注意兼容性, 使用上述第五条能起到相同作用)
  2. 使用ajax等技术异步刷新页面

# 其他手段(避免内存泄漏)

  1. 少使用全局变量
  2. 少使用闭包
  3. 及时清理无用定时器
  4. 及时清理无效的dom引用
